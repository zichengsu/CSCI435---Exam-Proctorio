# AI Exam Proctoring System (Phone & Person Detection)

## Overview
This project is an AI-based exam proctoring system that uses YOLOv8 to detect:
- Mobile phones
- Multiple people in frame

It helps ensure fairness in online examinations by detecting suspicious behavior in real time.

---

## Features
- Real-time webcam detection
- Phone detection using YOLOv8
- Multiple person detection
- Violation logging system
- Frame skipping for performance optimization

---

## Requirements
Install dependencies using:


pip install -r requirements.txt


---

## How to Run


python phone_detection.py


Press **q** to quit the program.
